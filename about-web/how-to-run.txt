How to Run 6gamer Locally

1. Prerequisites:
   - Node.js installed on your machine.
   - A Firebase project created on Firebase Console.
   - An SMTP service (like Gmail App Password) for sending OTPs.

2. Installation:
   - Extract the project files to a folder.
   - Open the terminal in that folder.
   - Run: npm install
   - This will install all dependencies including firebase, nodemailer, framer-motion, and lucide-react.

3. Configuration:
   - Create a .env.local file in the root directory.
   - Add your Firebase keys:
     NEXT_PUBLIC_FIREBASE_API_KEY=your_key
     NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_domain
     NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_id
     NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_bucket
     NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_id
     NEXT_PUBLIC_FIREBASE_APP_ID=your_id
   - Add your SMTP settings:
     SMTP_HOST=smtp.gmail.com
     SMTP_PORT=465
     SMTP_USER=your_email@gmail.com
     SMTP_PASS=your_app_password

4. Running the Project:
   - Run: npm run dev
   - Open http://localhost:3000 in your browser.

5. Error Solving:
   - If 'Firebase: Error (auth/invalid-api-key)' occurs, check your .env.local keys.
   - If OTP is not sending, verify your SMTP credentials and ensure 'Less Secure Apps' or 'App Passwords' are set up.
   - For 'Module not found' errors, run 'npm install' again.
